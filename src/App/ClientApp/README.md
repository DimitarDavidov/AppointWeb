# AppointWeb Frontend

React frontend for the AppointWeb application.

For setup instructions, documentation, and API reference, see the [project README](../../README.md) and [docs folder](../../docs/README.md).

## Quick start

```bash
npm install
npm run dev
```

Runs on http://localhost:5173 (requires the backend API on http://localhost:8080).

**Production:** set `VITE_API_URL` to the API URL before build. See [Deployment guide](../../docs/deployment.md).

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Dev server with hot reload |
| `npm run build` | Production build |
| `npm run lint` | ESLint |
| `npm run preview` | Preview production build |

See [Frontend documentation](../../docs/frontend.md) for folder structure, routing, and state management.
